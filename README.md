# DoLearn Frontend

Next.js frontend for DoLearn.

This app currently uses mock data and browser local storage while the backend API is being built separately in `../backend`.

## Local Setup

```bash
npm install
npm run dev
```

Open:

```txt
http://localhost:3000
```

## Scripts

```bash
npm run dev
npm run typecheck
npm run lint
npm run build
npm run start
```

## Deployment

Deploy this folder on Vercel. If using the full repo, set the Vercel root directory to `frontend`.
